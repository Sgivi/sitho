# Python Exercises README

This README file explains three simple Python exercises focusing on functions, conditionals, and returning values.


## 1. Rectangle Area Function

### Exercise Description

Create a function named `rectangle` that takes two parameters: `length` and `width`. The function calculates the area of a rectangle using the formula:

```
area = length * width
```

and returns the result.

### What It Does

This function takes two numbers (length and width), multiplies them, and returns the area of a rectangle.

## 2. Check if a Number is Even or Odd (Example 1)

### What It Does

This exercise defines a function that checks whether the number 79 is even or odd. It then prints the result.


## 3. Greeting Function

### What It Does
